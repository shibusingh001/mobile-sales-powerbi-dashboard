# Mobile Sales Dashboard (Power BI)
## 🎯Objectives
This project presents an interactive Mobile Sales Dashboard built using Power BI to analyze sales performance across different cities, brands, and time periods in India.
1.The dashboard provides a clear view of sales trends, customer behavior, and payment patterns to support data-driven decision-making.
2.Analyze total sales, quantity, and transactions
3.Identify top-performing mobile brands and models
4.Track monthly and daily sales trends
5.Understand customer ratings and satisfaction
6.Analyze preferred payment methods

## 📂 Dataset
The dataset used in this project is available in this repository:

🔗 [Mobile Sales Dataset](https://github.com/shibusingh001/mobile-sales-powerbi-dashboard/blob/main/Mobile%20Sales%20Data.xlsx)
- Format: Excel (.xlsx)
- Contains sales data including city, brand, model, quantity, and payment method

## 🛠Tools & Technologies
1.Power BI
2.DAX (Data Analysis Expressions) 
3.Excel / CSV Dataset

## 📊Key Metrics
💰 Total Sales: 769M
📦 Total Quantity Sold: 19K
📈 Average Sales: 40K
💳 Total Transactions: 4K

## 📈Dashboard Features
📍 Sales by City (Map Visualization)
📅 Monthly Sales Trend Analysis
📊 Sales by Mobile Model
⭐ Customer Ratings Distribution
💳 Transactions by Payment Method
📆 Sales by Day Name
🏷 Brand-wise Performance Table

## 🔍 Key Insights
📌 Sales peak observed in July and March
📌 UPI and Credit/Debit Cards dominate payment methods
📌 Top brands like Apple, Samsung, and OnePlus generate highest revenue
📌 Weekend sales (Saturday & Sunday) are higher than weekdays
📌 Certain cities contribute significantly to overall sales (e.g., metro cities)
